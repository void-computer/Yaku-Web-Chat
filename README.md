Web based chat app

you will need to install node.js to use this app

after installing node.js you need to make a new folder and use the following commands "npm init -y" , "npm instal express socket.io moment" and "npm install -D nodemon"

then delete the files package.json and package-lock.json

then copy and paste the files from the zip to the folder

then use the command "npm start server" or "npm run dev" to start the server then in you browser type in "localhost:8080 to use the website"
